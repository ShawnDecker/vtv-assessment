## Yoast SEO field-by-field setup

When pasting into WordPress, fill in:

| Yoast field | Maps to frontmatter |
|-------------|---------------------|
| SEO title | `yoast_seo_title` |
| Meta description | `yoast_meta_description` |
| Focus keyphrase | `yoast_focus_keyphrase` |
| Slug | `slug` |
| Featured image | use the `hero_image` description to source |
| Image alt text | `hero_image_alt` |
| Schema type | `schema` |

For each post, also set:
- Category (use the new taxonomy: Buying, Selling, Neighborhoods, Appraisal, Market Updates, Lifestyle — see SEO audit)
- Tags (3-5 specific neighborhood / topic tags)
- Internal linking from at least one previously-published post

## Publishing cadence recommendation

Don't publish all 10 at once. Cadence matters for:
- Google's crawl rhythm (publishing too fast looks spammy)
- Yoast cornerstone-content strategy
- Internal linking maturity (each post should link to 1-2 already-published posts)

| Week | Publish |

## After publishing each post

For maximum SEO leverage:

1. **Submit to Google Search Console** for indexing (manually request indexing on the URL)
2. **Share once on each social** with platform-appropriate copy (use `gergacs-post-writer` skill for variants)
3. **Email link to your top 5 newsletter subscribers** with a personal note ("thought you'd find this useful given what we discussed")
4. **Add internal links** from this new post to 2-3 already-published posts; from 1-2 already-published posts BACK to this new one
5. **Track in Notion Lighthouse** with: published date, URL, target keyphrase, current ranking (check weekly), traffic from Search Console
