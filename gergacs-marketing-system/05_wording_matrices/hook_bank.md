## How to test a hook

For every hook you ship:

1. **Persona check** — which of the 5 personas would stop scrolling for this? If "all," the hook is too generic.
2. **Specificity check** — does it name a place, a number, a feature, or a person? Generic = ignored.
3. **USPAP check** — does it imply a value guarantee? If yes, rewrite.
4. **Voice check** — would Nikki actually say this? Or does it sound like AI?
5. **Length check** — under 12 words for IG/TikTok. Under 60 chars for ad creative. Under 80 chars for blog title (for SEO).

## How to use this bank

- For a new piece, scan by structure first (Local Insider / Appraiser / Comparison / Relocator / Pearl Found / Honest Answer)
- Pick 2 hook options for A/B testing
- Customize the specifics (names, places, dollar bands) to the actual content
- Track which hooks pull (Monday Brief surfaces top-3 each week)
- Retire hooks that flop after 3 attempts, even if you like them

The hook is the only line that makes someone stop. Earn the stop.
