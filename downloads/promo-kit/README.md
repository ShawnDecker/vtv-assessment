## How to use the SVG cards

SVG files are scalable — they look perfect at any size. To use them on Instagram or TikTok:

1. **Open the SVG** in any browser, Figma, Canva, or vector tool.
2. **Export as PNG or JPG** at the displayed size:
   - 1080×1080 → IG feed square / X post
   - 1080×1920 → IG story / IG Reel cover / TikTok
   - 1200×630 → LinkedIn / FB / X link-card
3. **Customize if you want** (swap colors, add your logo, replace text — they're plain text SVGs).
4. **Upload** to the platform.

### Quick-export options
- **Online:** drop the SVG into [convertio.co](https://convertio.co) or [cloudconvert.com](https://cloudconvert.com)
- **Mac:** open in Preview → Export As PNG
- **Figma / Canva:** drag the SVG in → export at desired size
- **Command line (ImageMagick):** `magick card-01-hero-1080x1080.svg card-01-hero-1080x1080.png`
