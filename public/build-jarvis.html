Build Your Own Jarvis

Local voice assistant with wake-word, TTS, LLM replies, long-term memory, web search, database access, overnight design proposals, and Claude Code control. All open-source. CPU-only. No API keys.

1. Operating system and runtime

Windows 10 or 11 (64-bit). Linux works with minor path edits.

Python 3.12 — NOT 3.13 or 3.14. ML wheels lag the newest Python. Use py -3.12.

py -3.12 -m venv C:\Users\Cameron\jarvis\.venv
C:\Users\Cameron\jarvis\.venv\Scripts\activate
python -m pip install -U pip

3. Local AI — Ollama

Install: ollama.com/download/windows

ollama pull qwen3:8b          # 5.2 GB — main reasoning + SQL gen
ollama pull nomic-embed-text  # 275 MB — vector embeddings

Optional additions:

ollama pull phi4-mini:latest  # 2.5 GB — fastest butler replies
ollama pull gemma3:4b         # 3.3 GB — alternate reasoning

All run on CPU. Ollama exposes http://localhost:11434.

6. Windows-native wiring (no extra install)

  • Task Scheduler — overnight job at 11 pm / 2 am / 5 am
  • Startup folder — shell:startup shortcut, auto-launches on login
  • PowerShell — app launching and task registration

7. Optional — database access

Skip if not needed. For asking Jarvis "how many users" style questions from your own Postgres / Neon:

  1. Have your DB URL ready: postgresql://user:pass@host/db?sslmode=require
  2. Run setup_db_role.py once — creates a read-only jarvis_ro role, grants SELECT only, snapshots schema
  3. Voice queries like "how many members" then return live counts

8. What you do NOT need

  • No OpenAI or Anthropic API key
  • No Azure, Google Cloud, or AWS account
  • No microcontroller, Arduino, or sensor hardware
  • No GPU
  • No Docker (can containerize later)
  • No ElevenLabs or paid TTS subscription

9. First-run checklist

  • Install Python 3.12
  • Create venv at C:\Users\Cameron\jarvis\.venv
  • pip install the package list from Section 2
  • Install Ollama, pull qwen3:8b and nomic-embed-text
  • Copy the 15 Python files to C:\Users\Cameron\jarvis\
  • Edit config.yaml paths if username differs
  • Edit crawl_config.yaml to point at your project directories
  • Edit scenes.yaml for your workspaces (optional)
  • Run python jarvis.py --test-tts — verify voice
  • Run python jarvis.py --test-wake — say "Hey Jarvis", verify wake
  • Register Startup shortcut + Scheduled Task
  • (Optional) Run setup_db_role.py to wire Postgres
